基本数据同步时间

设置说明

1、应用会每天定时同步接口中的前一天的数据，并存储到本地。存储到本地之后，用户查询的日报数据从本地获取，不再从接口获取
2、数据同步时间说明：请根据医院实际情况，配置为数据已经完整的时间，否则当同步数据时，接口数据尚不完整，将会造成数据错误。默认配置时间为06:00，
     同步时间只能配置为整点
3、消息推送时间说明：请设置为晚于数据同步时间的时间点。默认配置为07:00，消息推送时间可设置为整点或半点
